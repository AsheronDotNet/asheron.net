Asheron[BOT] is our dedicated Discord server bot equipt with the following useful custom commands which are only available in our Discord channels. Launch Discord now to begin using him!

Launch Discord
!joinserver [server]

This command will tell Asheron to allow you access to be apart of and post in your Asheron's Call server channel(s). Simply type the command followed by one of the following accepted server names or initials. (Example: !joinserver Frostfell)

Darktide (DT)

Frostfell (FF)

Harvestgain (HG)

Leafcull (LC)

Morningthaw (MT)

Solclaim (SC)

Thistledown (TD)

Verdantine (VT)

Wintersebb (WE)

!acwiki [item]

In this command [item] refers to any of the pages located on the Asheron's Call Wiki website. For example using "!acwiki Asheron" will autmatically bring up the Wiki page for Asheron. Each result includes an image, description, and direct link to the specific page.
